Go Links Setup

⚠️ Setup Required: You must configure a base URL before Go Links will work.
Status: Not configured

Examples:

Company Wiki: https://wiki.company.com/

Development Server: https://localhost:3000/

Internal Tools: https://internal.company.com/tools/

How it works:

With base URL https://wiki.company.com/:

  • http://go/docs → https://wiki.company.com/docs
  • go api (in address bar) → https://wiki.company.com/api
